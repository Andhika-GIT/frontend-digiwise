# Digiwise ( Frontend )

Final capstone project of Generasi Gigih 3.0 Fullstack Track

Link to the website :  [Digiwise](https://digiwise.vercel.app/)

## **How to run in local**

**Pre-requisites**: Node, Docker

1. Clone the repo
2. Run the app using docker using the following command :

```
docker-compose up
```
